#### Specs:
- ESP32-S3-WROOM-1
- USB-C
- Pinouts for several transmitters for DIY users
- In-built E-Stop button and AUX E-Stop port (3.5mm TRS jack)
- No USB-UART, uses built-in USB on ESP32-S3
- 3.3v regulator
- Footprint 45.2mm x 75.7mm
