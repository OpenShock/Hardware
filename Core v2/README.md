#### Specs:
- ESP32-S3-WROOM-1
- USB-C
- In-built E-Stop button and AUX E-Stop port (3.5mm TRS jack)
- No USB-UART, uses built-in USB on ESP32-S3
- 3.3v regulator
- More compact than v1
- Footprint 40mm x 56mm
